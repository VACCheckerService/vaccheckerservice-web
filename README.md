## Worum geht's?

Cheater, die einen VAC-Ban kassiert haben, weichen oft einfach auf einen neuen, ungesperrten Account aus. VACCheckerService deckt genau das auf — durch den Abgleich von:

- 🔍 Steam-Profilen & Kontostatus
- 👥 Freundeslisten (aktuell & historisch)
- 🏷️ Gruppenmitgliedschaften
- 🖼️ Avatar-Historien
- 📛 Namens-Verläufen
- 🌐 Daten von [SteamID.uk](https://steamid.uk/)

Aus diesen Spuren entsteht eine nachvollziehbare Indizienkette, mit der Community-Admin-Teams handeln können.

## Ablauf

| Schritt | Was passiert |
|---|---|
| **1. Meldung** | Verdachtsfall wird über das Formular eingereicht (Profil-Link + Beschreibung) |
| **2. Analyse** | Freundeslisten, Gruppen, Avatare & Namen werden gegen bekannte VAC-Bans abgeglichen |
| **3. Beweis** | Dokumentierte Indizienkette geht an das Admin-Team zurück |
